# Contributing

Thanks for your interest in contributing to arca-router!

## Getting Started

- Search existing issues before opening a new one
- For non-trivial changes, please open an issue or discussion first to align on approach

## Developer Certificate of Origin (DCO)

To keep contributions easy for individuals and companies, we use a lightweight sign-off process.

By contributing, you agree that your work is submitted under the project license and that you have the right to submit it.

Please sign off your commits:

```bash
git commit -s
```

## Pull Requests

- Keep PRs focused and small when possible
- Include tests or reproduction steps where relevant
- Avoid including secrets or private keys in configs/logs
